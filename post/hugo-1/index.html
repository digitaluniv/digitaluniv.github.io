Hugo をインストールしてして、Webページを作成するところまで試します。

Webサイト構築

Hugoをインストールして、Webサイトを構築します。

Hugoのインストール

HomebrewでHugoとtreeコマンドをインストールします。

$ brew install hugo
$ brew install tree

Hugoのサイト構築

「hugo_site」という名前のサイトを作ります。。

$ hugo new site hugo_site
$ cd hugo_site
$ tree
.
├── archetypes
│   └── default.md
├── config.toml
├── content
├── data
├── layouts
├── static
└── themes

テーマの導入

公式サイトからサイトのテンプレートを選びます 今回は「Whiteplain」を選びました。

$ git clone https://github.com/taikii/whiteplain.git themes/whiteplain

サイトの設定

サイトの各種設定をします。

Configration

Config.tomlを編集します。 このサイトの設定例を掲載します。

baseurl = "https://digitaluniv.github.io/"
title = "Digital University"
theme = "whiteplain"
languageCode = "ja-jp"
hasCJKLanguage = true

copyright = "&copy; Copyright 2018 digitaluniv"

[params]

#dateFormat = "January 2, 2006"

[params.author]
name = "digitaluniv"
avatar = "/digitaluniv.jpg"
github = "digitaluniv"

[taxonomies]
category = "categories"
tag = "tags"
series = "series"

[[menu.main]]
name = "Categories"
identifier = "categories"
url = "/categories/"
weight = 1
[[menu.main]]
name = "Tags"
identifier = "tags"
url = "/tags/"
weight = 2
[[menu.main]]
name = "Archives"
identifier = "archives"
url = "/archives/"
weight = 3
[[menu.main]]
name = "About"
identifier = "about"
url = "/about/"
weight = 4

[[menu.footer]]
name = "About"
weight = 1
identifier = "about"
url = "/about/"

Archives page

contentフォルダ内にarchivesフォルダを作り空の_index.mdファイルを作ります。

hugo_site
└── content
    └── archives
        └── _index.md

About page

contentフォルダ内にaboutフォルダを作り_index.mdファイルにこのサイトの概要を書きます。

hugo_site
└── content
    └── about
        └── _index.md

avator

作者のアバターを avatar = “/digitaluniv.jpg” で指定しましたので、 staticフォルダーに画像を入れておきます。

hugo_site
└── static
    └── digitaluniv.jpg

記事の雛形

archetypesフォルダのdefolt.mdを編集します。

---
title: "{{ replace .TranslationBaseName "-" " " | title }}"
date: {{ .Date }}
draft: true
categories: ["x", "y"]
tags: ["x", "y"]
---

記事の作成と確認

記事を作成してWebサイトをローカルで確認します。

記事の作成

hugo_siteのフォルダー内に移り、hugo new post/hoge.mdコマンドで記事を作成します。

ここでは、test.mdというファイルを作りました。

$ hugo new post/test.md

---
title: "Test"
date: 2018-02-03T00:04:17+09:00
draft: true
categories: ["x", "y"]
tags: ["x", "y"]
---

categoriesやtagsをセットして、本文はマークダウン形式で書いていきます。

記事の確認

ローカルで記事の確認を行います。

$ hugo server -D

-Dオプションでドラフト記事もビルドしてくれます。

記事の生成

記事が完成しましたら、draft: trueを外します。

$ hugo

これで記事が生成されました。
